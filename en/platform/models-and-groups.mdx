## What Are Groups

RuoLi uses **groups** to organize different model channels. Each group corresponds to a different upstream API source with its own pricing multiplier.

When creating a token, you need to select a group. The group determines the **pricing** and **available channels** for your API calls.

## What Does Multiplier Mean

The multiplier is a pricing coefficient applied on top of the model's official price. **The lower the multiplier, the cheaper it is.**

Actual cost = **Model official price x Group multiplier**

For example, if a model's official price is 10 yuan per million tokens:
- Using `claude-officially` (multiplier 1.6): cost = 10 x 1.6 = **16 yuan**
- Using `claude-ant` (multiplier 0.15): cost = 10 x 0.15 = **1.5 yuan**
- Using `free` (multiplier 0): **completely free**

Different models have different official prices within the same group, so the final cost varies. See [Billing & Top-Up](/en/platform/billing) for details.

## How to Switch Groups

Groups are bound to **tokens**. To switch groups, simply create a new token and select a different group.
