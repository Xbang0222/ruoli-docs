## Step 1: Create an Account

Go to [ruoli.dev](https://ruoli.dev) to register your account and log in.

## Step 2: Create an API Key (Token)

1. Go to the console and click the **Tokens** page
2. Click **Add Token**
3. Enter a token name
4. Select a **Group** (determines the pricing multiplier — see [Models & Groups](/en/platform/models-and-groups))
5. Select available models as needed (leave empty = all models)
6. Click **Create** and copy the generated API Key

<Warning>The API Key is only shown once — save it immediately. If lost, you'll need to create a new one.</Warning>
