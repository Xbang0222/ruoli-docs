---
title: "本地开发"
description: "在本地预览文档更改"
---

<Info>
  **前置条件**：
  - Node.js 19 或更高版本
  - 包含 `docs.json` 文件的文档仓库
</Info>

按照以下步骤在你的操作系统上安装和运行 Mintlify。

<Steps>
<Step title="安装 Mintlify CLI">

```bash
npm i -g mint
```
</Step>

<Step title="本地预览">

进入 `docs.json` 所在的文档目录，运行以下命令：

```bash
mint dev
```

本地文档预览将在 `http://localhost:3000` 可用。

</Step>
</Steps>

## 自定义端口

默认使用 3000 端口。使用 `--port` 参数可以自定义端口：

```bash
mint dev --port 3333
```

## 验证链接

CLI 可以帮助验证文档中的链接。使用以下命令检查断开的链接：

```bash
mint broken-links
```
