## 第 1 步：注册账号

前往 [ruoli.dev](https://ruoli.dev) 注册你的账号并登录。

## 第 2 步：创建 API Key（令牌）

1. 进入控制台，点击 **令牌** 页面
2. 点击 **添加令牌**
3. 填写令牌名称
4. 选择**分组**（决定价格倍率，详见[模型与分组](/platform/models-and-groups)）
5. 按需选择可用模型（留空 = 全部模型）
6. 点击 **创建**，复制生成的 API Key
