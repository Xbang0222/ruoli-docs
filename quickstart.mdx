## 三步上手

搭建本地文档站点并进行第一次自定义。

### 第 1 步：设置本地环境

<AccordionGroup>
  <Accordion icon="rectangle-terminal" title="启动预览服务器">
    1. 安装 Mintlify CLI：`npm i -g mint`
    2. 进入文档目录并运行：`mint dev`
    3. 打开 `http://localhost:3000` 查看文档预览

    <Tip>编辑文件后预览会自动更新。</Tip>
  </Accordion>
</AccordionGroup>

### 第 2 步：部署更改

<Accordion icon="github" title="安装 GitHub App">
  从 [dashboard](https://dashboard.mintlify.com/settings/organization/github-app) 安装 Mintlify GitHub App。

  推送到默认分支后，更改会自动部署到生产环境。
</Accordion>

### 第 3 步：上线

<Accordion icon="rocket" title="发布文档">
  1. 提交并推送你的更改。
  2. 文档将在片刻后更新并上线。
</Accordion>
